\paragraph{Ejemplo de Código en Python}
Aquí hay un ejemplo de cómo se podría incluir código Python utilizando el paquete \texttt{listings}:

\begin{lstlisting}[language=Python, caption=Ejemplo de función para análisis de datos, label=lst:data-analysis]
import pandas as pd

def analyze_data(file_path):
    """Función para leer y analizar datos de un archivo CSV."""
    data = pd.read_csv(file_path)
    
    # Estadísticas básicas
    summary = data.describe()
    
    # Análisis de valores faltantes
    missing = data.isnull().sum()
    
    return summary, missing
\end{lstlisting}
